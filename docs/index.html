Professional Auto Liability Insurance Rating Engine

Desktop application for calculating commercial auto liability premiums from quote PDFs. Supports both GUI and CLI modes with complete offline operation.

Key Features

Dual-Mode Operation

Use the intuitive desktop GUI or powerful command-line interface - your choice!

Smart PDF Parsing

Automatic text extraction with OCR fallback for image-based PDFs (80% confidence threshold).

Factor-Based Rating

Precise calculations using: Base_AL × F_vehicle × F_driver × F_radius × F_limit

Reconciliation

Automatic comparison with printed totals using configurable $0.50 tolerance.

Audit Trail

Complete calculation history stored in SQLite with JSON export capabilities.

Offline-First

No internet required. All operations work completely offline.

Multi-State Support

Automatic CW/SS program selection with state-specific tax configurations.

Security Built-In

Macro-enabled workbook detection and rejection for security compliance.

Technical Architecture

Core Components

  • Rating Engine: Factor-based calculation algorithm
  • PDF Parser: pdfplumber → PyMuPDF → Tesseract OCR
  • Data Loaders: Excel rating tables & tax configs
  • Storage: SQLite database + JSON export
  • UI: NiceGUI desktop framework

Rating Algorithm

rate_per_unit = Base_AL × F_body_class × F_radius × F_driver × F_limit AL_Total = Premium_Subtotal + Fees_Total + Taxes_Total

Technology Stack

  • Language: Python 3.11+
  • PDF Processing: pdfplumber, PyMuPDF, pytesseract
  • Excel: openpyxl, pandas
  • UI: NiceGUI
  • Database: SQLite3
  • Testing: pytest (73 tests, 85% coverage)

Quick Example - CLI Usage

# Launch desktop GUI $ romans-rater # Process a PDF via CLI $ romans-rater quote.pdf # Use OCR for image-based PDFs $ romans-rater quote.pdf --ocr # View results AL Premium Subtotal: $7,250.00 Fees Total: $ 225.00 Taxes Total: $ 276.13 ───────────────────────────────── AL Total: $7,751.13 ✓ Reconciliation: match ($-0.25)
